## 🎯 产品定位

**AI智能标书生成平台**是一款基于人工智能的投标文档自动化处理系统,通过AI技术帮助企业快速、准确地完成标书制作,大幅提升投标效率和中标率。

**核心价值**:
- 🚀 **效率提升**: 标书制作时间从7天缩短至1-2天
- 🎯 **质量保证**: AI辅助生成,人工审核,确保质量
- 📚 **知识沉淀**: 企业知识库持续积累,越用越智能
- 🤖 **智能应答**: 自动匹配需求,生成定制化应答文档

   - 本月中标数 (Won This Month)

   - AI生成任务完成通知
   - 审核提醒
   - 截止日期提醒

**数据来源**:
- `/api/projects/statistics` - 获取统计数据
- `/api/projects?status=in_progress` - 获取进行中的项目列表
- `/api/activities/recent` - 获取最近活动记录

## 2️⃣ 项目管理 (Project Management)

### 定位
专注于**投标项目的生命周期管理**,是工作的核心驱动引擎。

### 核心功能

#### 2.1 投标管理 (Tender Management / HITL系统)

**功能描述**:
这是所有投标项目的**中枢枢纽**,用户可以在这里创建新项目、跟踪项目阶段、分配任务、归档项目等。AI工具的处理结果最终也会关联到这里的具体项目中。

**功能模块**:

1. **项目列表**
   - 显示所有项目
   - 状态筛选:全部、进行中、已中标、未中标、已归档
   - 支持搜索(项目名称、编号)
   - 批量操作:归档、删除

2. **项目详情**
   - 基本信息:项目名称、编号、客户、预算、截止日期
   - 项目阶段:投标准备、文档制作、审核、提交、评审、结果
   - 关联文档:招标文档、商务应答、技术方案、点对点应答
   - 任务列表:待办事项、责任人、进度
   - 操作日志:所有操作记录

3. **HITL人机协同工作流**
   - Step 1: 招标文档解析
   - Step 2: 需求提取和审核
   - Step 3: AI生成文档(商务应答、点对点、技术方案)
   - Step 4: 人工审核和修改
   - Step 5: 文档融合和导出

4. **项目创建向导**
   - 第1步:填写项目基本信息
   - 第2步:上传招标文档
   - 第3步:选择要生成的文档类型
   - 第4步:启动AI处理

**数据来源**:
- `/api/projects` - 项目CRUD操作
- `/api/hitl/{hitl_task_id}` - HITL任务详情
- `/api/tender-processing/start` - 启动文档处理

## 3️⃣ AI核心工具 (Core AI Tools)

### 定位
集合所有用于**"执行"具体投标任务的智能化工具**。用户既可以从【项目管理】中为特定项目调用这些工具,也可以独立使用它们来完成临时任务。

### 核心功能

#### 3.1 智能应答 (Intelligent Response)

##### 3.1.1 商务应答 (Business Response)

**功能描述**:
处理标书中的**商务条款、公司资质、报价**等部分的应答。AI自动从企业库中匹配资质文件,生成符合招标要求的商务应答文档。

**核心流程**:
```
上传招标文档 → AI提取商务要求 →
匹配企业资质 → 生成应答文档 →
人工审核 → 导出Word文档
```

**功能模块**:
1. 文档上传:支持PDF、Word格式
2. 需求提取:AI自动提取商务要求(营业执照、资质证书、财务报表等)
3. 资质匹配:从企业库自动匹配符合要求的资质文件
4. 应答生成:AI生成应答文本,插入资质文件图片
5. 人工审核:用户可编辑、替换、删除内容
6. 文档导出:生成Word文档,保留格式

**数据来源**:
- `/api/business-response/generate` - 生成商务应答
- `/api/companies/{company_id}/qualifications` - 获取企业资质
- `/api/business-response/stream/{task_id}` - SSE实时进度

##### 3.1.2 点对点应答 (Point-to-Point Response)

**功能描述**:
针对技术需求清单进行**逐条应答**。AI根据需求关键词从知识库中检索相关文档,生成针对性的应答内容。

**核心流程**:
```
上传需求清单 → AI逐条分析 →
知识库检索 → 生成应答内容 →
人工审核 → 导出表格/文档
```

**功能模块**:
1. 需求清单上传:支持Excel、Word表格
2. 需求解析:AI识别每条需求的关键词
3. 知识检索:从文档库、案例库中检索相关内容
4. 应答生成:逐条生成应答(符合/不符合/部分符合)
5. 批量编辑:支持批量修改应答内容
6. 应答导出:生成Excel表格或Word文档

**数据来源**:
- `/api/point-to-point/generate` - 生成点对点应答
- `/api/knowledge/search` - 知识库检索
- `/api/point-to-point/stream/{task_id}` - SSE实时进度

#### 3.2 方案生成 (Solution Generation)

##### 3.2.1 技术方案 (Technical Solution)

**功能描述**:
根据需求和知识库内容,智能生成**完整的技术方案文档**。AI自动规划章节结构、填充内容、插入案例和产品信息。

**核心流程**:
```
选择章节模板 → 上传招标文档 →
AI生成大纲 → 智能填充内容 →
插入案例/产品 → 人工审核 → 导出Word
```

**功能模块**:
1. 章节模板:预定义的标书章节结构(可自定义)
2. 智能大纲:AI根据招标需求生成章节大纲
3. 内容生成:从知识库提取内容,AI改写和组织
4. 产品组合:自动选择合适的产品线组合进方案
5. 案例插入:从案例库匹配相似项目案例
6. 图表生成:AI生成架构图、拓扑图(未来功能)
7. 方案导出:生成Word文档,支持自定义模板

**高级特性 - 产品组合**:
- AI智能推荐公司的不同产品线组合
- 根据招标需求自动匹配最优产品配置
- 支持手动调整产品组合方案

**数据来源**:
- `/api/tech-proposal/generate-outline` - 生成技术方案大纲
- `/api/tech-proposal/fill-content` - 填充章节内容
- `/api/knowledge/documents/search` - 知识库文档检索
- `/api/cases/search` - 案例库检索
- `/api/tech-proposal/stream/{task_id}` - SSE实时进度

#### 3.3 智能评审 (Intelligent Review)

##### 3.3.1 标书评分 (Tender Scoring)

**功能描述**:
上传标书或评分标准,**AI辅助进行风险分析和预估得分**,帮助企业决策是否参与投标。

**核心流程**:
```
上传招标文件 → AI提取评分标准 →
自评打分 → 风险分析 →
生成报告 → 决策建议
```

**功能模块**:
1. 评分标准提取:AI自动识别评分项和权重
2. 自动评分:根据企业资质和能力自动打分
3. 差距分析:对比企业能力与招标要求,识别短板
4. 风险评估:分析中标概率和潜在风险
5. 决策建议:AI给出是否参与投标的建议
6. 报告生成:生成评分报告和分析文档

**数据来源**:
- `/api/tender-scoring/analyze` - 分析招标文档
- `/api/tender-scoring/score` - 自动评分
- `/api/companies/{company_id}/capabilities` - 获取企业能力评估

## 4️⃣ 知识中心 (Knowledge Center)

### 定位
整个AI系统的**"大脑"和"资料库"**,为所有AI工具提供数据支持。知识库越完善,AI生成的内容质量越高。

### 核心功能

#### 4.1 企业库 (Company Library)

**功能描述**:
管理公司的**基本信息、资质证书、荣誉奖项、财务状况**等。这是"商务应答"模块的主要数据来源。

**功能模块**:

1. **企业基本信息**
   - 公司名称、统一社会信用代码
   - 注册地址、经营范围
   - 法定代表人、注册资本
   - 成立日期、联系方式

2. **资质证书管理** (17+种标准资质)
   - 营业执照
   - ISO 9001质量管理体系认证
   - ISO 27001信息安全管理体系认证
   - ISO 14001环境管理体系认证
   - ISO 45001职业健康安全管理体系认证
   - CMMI能力成熟度模型集成
   - 软件企业认定证书
   - 高新技术企业证书
   - 等保三级备案证明
   - AAA信用等级证书
   - 行业特定资质(建筑、医疗、教育等)
   - 其他自定义资质

3. **荣誉奖项**
   - 政府奖项
   - 行业奖项
   - 客户表彰

4. **财务信息**
   - 年营业额
   - 纳税证明
   - 审计报告
   - 股东信息
   - 是否有管理公司

5. **企业文化**
   - 企业介绍
   - 发展历程
   - 企业愿景和使命

**数据来源**:
- `/api/companies` - 企业CRUD操作
- `/api/companies/{company_id}/qualifications` - 资质管理
- `/api/qualification-types` - 资质类型列表

#### 4.2 案例库 (Case Library)

**功能描述**:
存储和管理公司**过往的成功项目案例、解决方案和客户评价**。主要用于"技术方案生成"时丰富方案内容,提升说服力。

**功能模块**:

1. **案例基本信息**
   - 项目名称
   - 客户名称
   - 项目规模(金额、周期、人员)
   - 项目类型(行业、技术领域)
   - 项目时间

2. **案例内容**
   - 项目背景和需求
   - 解决方案描述
   - 技术架构
   - 实施过程
   - 项目成果
   - 客户评价

3. **案例附件**
   - 项目文档
   - 架构图
   - 实施照片
   - 验收报告

4. **案例标签**
   - 行业标签(政府、金融、教育、医疗等)
   - 技术标签(云计算、大数据、AI、物联网等)
   - 规模标签(大型、中型、小型)

5. **案例检索**
   - 关键词搜索
   - 标签筛选
   - 相似案例推荐

**数据来源**:
- `/api/cases` - 案例CRUD操作
- `/api/cases/search` - 案例检索
- `/api/cases/{case_id}/files` - 案例附件管理

#### 4.3 文档库 (Document Library)

**功能描述**:
通用的**文档资料库**。存放产品白皮书、技术手册、标准操作流程(SOP)、解决方案模板等可复用文档。

**功能模块**:

1. **文档分类**
   - 产品文档(产品手册、白皮书、技术规格)
   - 解决方案(行业解决方案、技术方案模板)
   - 标准文档(SOP、管理制度、质量标准)
   - 培训材料(培训课件、操作手册)
   - 其他文档

2. **文档管理**
   - 上传文档(PDF、Word、PPT、Excel)
   - 文档预览
   - 版本管理
   - 文档标签

3. **知识检索**
   - 全文搜索(基于FAISS向量搜索)
   - 语义检索(AI理解用户意图)
   - 相关文档推荐

4. **文档引用**
   - 跟踪文档被引用情况
   - 统计使用频率
   - 优化知识库质量

**数据来源**:
- `/api/knowledge/documents` - 文档CRUD操作
- `/api/knowledge/search` - 向量检索
- `/api/knowledge/upload` - 文档上传和向量化

#### 4.4 简历库 (Resume Library)

**功能描述**:
管理**项目团队成员和技术专家的简历**。在制作标书时,可快速筛选并附上符合项目要求的核心成员简历。

**功能模块**:

1. **人员基本信息**
   - 姓名、性别、年龄
   - 学历、专业
   - 职称、职务
   - 联系方式

2. **工作经历**
   - 工作单位
   - 工作时间
   - 职位描述

3. **项目经验**
   - 项目名称
   - 项目角色
   - 项目时间
   - 项目描述
   - 技术栈

4. **技能证书**
   - 专业证书(PMP、软考、云认证等)
   - 技能评级
   - 证书有效期

5. **简历模板**
   - 多种简历模板
   - 自定义模板
   - 批量生成简历

6. **简历筛选**
   - 按技能筛选
   - 按经验筛选
   - 按可用性筛选
   - 智能推荐(根据项目需求)

7. **简历导出**
   - 单个简历导出(Word/PDF)
   - 批量导出
   - 附加到标书

**数据来源**:
- `/api/resumes` - 简历CRUD操作
- `/api/resumes/search` - 简历检索
- `/api/resumes/export` - 简历导出
- `/api/resumes/{resume_id}/projects` - 项目经验管理

## 📊 使用场景举例

### 场景1: 新项目投标完整流程

1. **工作台** → 点击"开始投标"快捷入口
2. **项目管理** → 创建新项目,上传招标文档
3. **HITL工作流** → AI解析招标文档,提取需求
4. **商务应答** → AI自动匹配企业资质,生成商务应答
5. **点对点应答** → AI逐条应答技术需求清单
6. **技术方案** → AI生成技术方案大纲,填充内容
7. **人工审核** → 用户审核、修改AI生成的内容
8. **文档融合** → 将所有文档合并为完整标书
9. **导出提交** → 导出最终标书,准备提交

### 场景2: 知识库日常维护

1. **知识中心** → 进入"企业库"
2. 上传新获得的资质证书(如ISO 27001)
3. 进入"案例库",添加最近完成的项目案例
4. 进入"文档库",上传新产品的技术白皮书
5. 进入"简历库",更新团队成员的项目经验

**效果**: 知识库越完善 → AI生成的内容质量越高 → 中标率提升
